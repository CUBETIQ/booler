# Booler: Convert any values to Boolean.

Convert string/values to Boolean in JavaScript.

## Installing

```shell
$ npm i booler.js --save
```

## Example

```js
const booler = require('booler')

console.log(booler('f')) // return false
console.log(booler('t')) // return true
```

## Run Test

```shell
$ npm run test
```
